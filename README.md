## ğŸ“¦ Installation

### Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14.0.0 or higher)
- [npm](https://www.npmjs.com/) (v6.0.0 or higher) or [yarn](https://yarnpkg.com/)

### Step-by-Step Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/cu-campusbid.git
   cd cu-campusbid
   ```

2. **Create React App** (if starting fresh)
   ```bash
   npx create-react-app cu-campusbid
   cd cu-campusbid
   ```

3. **Install dependencies**
   ```bash
   npm install lucide-react
   ```

4. **Add the project files**
   - Copy `AuctionPlatform.jsx` to `src/AuctionPlatform.jsx`
   - Copy `AuctionPlatform.css` to `src/AuctionPlatform.css`

5. **Update `src/App.js`**
   ```javascript
   import React from 'react';
   import AuctionPlatform from './AuctionPlatform';

   function App() {
     return <AuctionPlatform />;
   }

   export default App;
   ```

6. **Start the development server**
   ```bash
   npm start
   ```

7. **Open your browser**
   
   Navigate to `http://localhost:3000`
